# E-commerce BackEnd ORM

## Description
Internet retail, also known as e-commerce, is the largest sector of the electronics industry, having generated an estimated US$29 trillion in 2017 (Source: United Nations Conference on Trade and Development).Here is the backend of an e-commerce site. It takes a working Express API and is configured to use Sequelize to interact with a MySQL database.

## Installation
We use MySQL2 and Sequelize to connect to Express.js API to MySQL database.
To Seed the database Run ```npm run seed```. 
Run ```npm start``` to start up the server.
